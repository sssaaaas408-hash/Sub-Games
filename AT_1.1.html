Grande Atualização: Nova Interface e Aplicativo!

A versão 1.1 traz mudanças significativas para o SubGames! Renovamos completamente a interface web, lançamos o aplicativo oficial e implementamos diversos recursos que a comunidade pediu. Esta é a maior atualização desde o lançamento da plataforma!

Nova Interface Web

Redesenhamos completamente a interface para proporcionar uma experiência mais moderna e intuitiva

Novidades Principais

Nova Interface Web Design completamente renovado com melhor organização, cards mais atraentes e experiência de usuário aprimorada
Nova Logo Oficial Logo redesenhada profissionalmente por zark2.0_, trazendo uma identidade visual moderna e marcante para o SubGames
Lançamento do Aplicativo Aplicativo oficial do SubGames disponível para download com todas as funcionalidades da versão web
Sistema Online Fix Integrado Online Fix agora disponível diretamente dentro do aplicativo, facilitando a ativação de jogos multiplayer
Sistema de Configurações Novo painel de configurações no aplicativo para personalizar sua experiência
Sistema de Notificações Alertas automáticos dentro do app sempre que houver novas atualizações disponíveis
Nova Base de Dados Migração para infraestrutura mais robusta e rápida, melhorando velocidade de downloads
Novo Servidor Discord Servidor Discord dedicado exclusivamente ao SubGames com canais organizados e suporte dedicado

Melhorias e Otimizações

Remoção da Aba Biblioteca Removemos a aba Biblioteca para simplificar a navegação, focando na experiência de download direto
Performance Aprimorada Carregamento de páginas até 40% mais rápido e melhor gestão de memória no aplicativo
Interface Mais Responsiva Melhor adaptação para diferentes tamanhos de tela e dispositivos móveis
Sistema de Busca Melhorado Busca mais rápida e precisa com sugestões em tempo real

Correções de Bugs

Corrigido problema de downloads interrompidos Downloads agora são mais estáveis e menos propensos a falhas
Resolvido erro de autenticação Problemas com login do Discord foram completamente resolvidos
Corrigido tema escuro em certos elementos Todos os elementos agora respeitam corretamente o tema selecionado
