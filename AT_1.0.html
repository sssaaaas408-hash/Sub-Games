Lançamento Beta do SubGames!

Estamos muito felizes em anunciar o lançamento oficial da versão Beta do SubGames! Esta é a primeira versão da nossa plataforma de download de jogos, criada para proporcionar uma experiência rápida, segura e intuitiva para toda a comunidade.

O SubGames nasceu com o objetivo de facilitar o acesso a manifests de jogos de forma simples e direta, sem complicações. Estamos comprometidos em melhorar constantemente a plataforma com base no feedback da comunidade.

Funcionalidades Implementadas

Sistema de Login Discord Autenticação segura através do Discord para acesso à plataforma
Download de Manifests Sistema de busca e download de arquivos ZIP por código único
Modo Escuro/Claro Toggle para alternar entre temas claro e escuro, com preferência salva
Design Responsivo Interface adaptável para desktop, tablet e mobile com menu hamburger
Base de Dados em Nuvem Integração com Supabase para armazenamento e gestão de arquivos
Interface Moderna Design limpo e profissional inspirado em plataformas modernas
Performance Otimizada Carregamento rápido e downloads eficientes
Sistema de Busca por Código Busca rápida de jogos através de códigos únicos

Próximos Passos

Biblioteca de Jogos Catálogo completo com todos os jogos disponíveis
Sistema de Comentários Permitir que usuários avaliem e comentem sobre os jogos
Sistema de Favoritos Salvar jogos favoritos para acesso rápido
Notificações Alertas sobre novos jogos e atualizações

Feedback: Esta é uma versão Beta, e sua opinião é fundamental! Entre no nosso servidor Discord e compartilhe suas sugestões, reporte bugs ou simplesmente converse com a comunidade. Juntos, vamos tornar o SubGames ainda melhor!
