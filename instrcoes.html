Como Baixar Jogos

Siga este guia passo a passo para começar a baixar seus jogos favoritos usando o SubGames

1
Baixar o Steam Tools
Primeiro, você precisa baixar e instalar o Steam Tools. Esta ferramenta é essencial para gerenciar os manifests dos jogos.
Baixar Steam Tools
Após baixar, extraia o arquivo e execute o Steam Tools. Certifique-se de ter o Steam instalado no seu computador.
2
Baixar o ZIP do Jogo
Use a página Web do SubGames para baixar o arquivo ZIP contendo os manifests do jogo que você deseja. Digite o código do jogo e escolha a versão SAN 1.0 ou 1.1 BETA.
Ir para Downloads
Após o download, extraia o arquivo ZIP em uma pasta de fácil acesso. Você verá vários arquivos .manifest dentro da pasta.
3
Arrastar os Manifests
Abra o Steam Tools e arraste todos os arquivos .manifest que você extraiu do ZIP para dentro da janela do Steam Tools. Os arquivos serão automaticamente reconhecidos.
Certifique-se de arrastar TODOS os arquivos .manifest de uma vez. O Steam Tools irá detectá-los e prepará-los para download.
4
Reiniciar o Steam Tools
Após adicionar os manifests, feche completamente o Steam Tools e abra novamente. Isso garantirá que todos os arquivos sejam carregados corretamente e o download possa começar.
Após reiniciar, os jogos devem aparecer na sua lista e começar a baixar automaticamente. Aguarde o download finalizar e aproveite!

Pronto!

Agora você já sabe como baixar jogos usando o SubGames. Se tiver alguma dúvida, visite nossa seção de suporte ou entre no nosso servidor Discord para obter ajuda da comunidade.
